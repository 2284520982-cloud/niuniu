# 🎯 开箱即用说明

## 一键启动

### Windows用户
**双击 `start.bat` 文件即可！**

### Linux/Mac用户
```bash
chmod +x start.sh
./start.sh
```

### 或使用Python
```bash
python start.py
```

## 启动后

1. **等待服务启动**（首次运行会自动安装依赖）
2. **浏览器打开**：http://localhost:7777/static/index.html
3. **开始使用**：填写项目路径，点击"开始扫描"

## 默认配置

所有配置都已设置好，可以直接使用：

- ✅ 规则文件：自动加载（Rules/rules.json + 所有增强规则）
- ✅ 扫描深度：15层（已优化）
- ✅ 模板扫描：开启
- ✅ Lite富化：开启（推荐）
- ✅ 端口：7777

## 无需配置

项目已经配置好，**无需任何额外设置**即可使用！
